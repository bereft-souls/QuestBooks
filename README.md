# QuestBooks

---

QuestBooks is a mod that adds in-game quest books to Terraria!

Far too long have players had to over-rely on having 5 wiki pages open at a time. Time spent in a web browser is time taken away from experiencing the content in a fun, natural manner. This is where QuestBooks comes in.

While wiki dependence may never fully disappear, QuestBooks eliminates a vast majority of the necessity to have a page open at all times. The integrated quest lines are organic and natural, guiding players through the game's content without directly holding their hand, while still leaving plenty of intricate sub-content to find on their own!

---

## API / Usage

QuestBooks also contains an open-source API to allow other mod developers to implement their own quest lines into the game!

Check out the [setup documentation](https://github.com/bereft-souls/QuestBooks/blob/master/docs/SETUP.md) to get started!
